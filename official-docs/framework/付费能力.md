# 付费能力

来源: https://developers.weixin.qq.com/miniprogram/dev/platform-capabilities/charge/

#
付费管理
小程序有丰富的开放能力，由于技术服务与运营成本原因，部分能力需付费使用，开发者可根据小程序自身业务场景需要而选择购买。
付费管理模块，承载开发者在小程序里购买各项付费能力，管理订单和用量，开具发票的功能。
#
一、了解各项付费能力
1.
登录小程序
MP平台
；
2.
点击左侧菜单「付费管理」；
3.
查看可以购买的各项能力、用量；
4.
查看文档或查看详情，了解各项能力用法，以及计费规则。
#
二、购买付费能力
1.
在“能力购买”里，选择所需购买的能力，点击“购买”；
2.
参考【最近三个月平均用量】，选择所需购买的资源包。
3.
点击【支付】，完成付款。支持微信支付和网银支付两种方式，如需对公账户付款，可以使用网银支付方式。后续将支持线下打款等更多付款方式。
#
三、查看订单、开具发票
1.
在「我的订单」中可查看订单列表，并开具发票；
2.
在「发票」中可查看开过的发票列表；
3.
可以申请增值税专用发票，将通过邮寄方式送达；
4.
发票开具时间预计为5-10个工作日。
#
四、查看用量
在「我的用量」中可查看各项能力的用量情况
#
五、用量提醒
当资源包将要到期、将要用完时，平台将对小程序管理员、运营人员发送用量预警的消息通知，请及时关注和补充用量。
平台将在资源包余量不足20%、10%、5%及用完时发送模板消息预警。
