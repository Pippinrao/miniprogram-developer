#
预设路由
为降低开发成本，基础库预设了一批常见的路由动画效果。
routeType
最低基础库版本
`wx://bottom-sheet`
3.1.0
`wx://upwards`
3.1.0
`wx://zoom`
3.1.0
`wx://cupertino-modal`
3.1.0
`wx://cupertino-modal-inside`
3.1.0
`wx://modal-navigation`
3.1.0
`wx://modal`
3.1.0
#
使用方法
仅需在路由跳转时，指定对应的
`routeType`
。
注: 以上路由效果均可通过
自定义路由
实现，可参考示例代码中的源码文件，自定义所需效果。

```
wx.navigateTo({
  url: 'xxx',
  routeType: 'wx://modal'
})

```
