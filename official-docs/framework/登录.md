# wx.login 登录

来源: https://developers.weixin.qq.com/miniprogram/dev/framework/open-ability/login.html

## 概述

小程序通过 `wx.login` 获取临时 code，后端用 code 换取用户的 `openid` 和 `session_key`，实现用户身份认证。

## 流程

```
小程序端                    微信服务端              开发者后端
  │                           │                      │
  ├─ wx.login() ──────────────►                      │
  │                           │                      │
  ◄── code ──────────────────┤                      │
  │                           │                      │
  ├───── code ─────────────────────────────────────► │
  │                           │                      │
  │                           ◄── code + appid +     │
  │                           │    app_secret ───────┤
  │                           │                      │
  │                    ┌──────┤── openid ────────────►
  │                    │      │   session_key        │
  │                    │      │                      │
  │                    │      │  ◄── 自定义登录态 ───┤
  │                           │                      │
  ◄── 自定义登录态(token) ─────────────────────────┤
```

## wx.login 调用

```javascript
wx.login({
  success(res) {
    if (res.code) {
      // 将 code 发送到后端换取 openid 和 session_key
      wx.request({
        url: 'https://your-server.com/api/login',
        data: { code: res.code },
        success(response) {
          // 保存自定义登录态
          wx.setStorageSync('token', response.data.token);
        }
      });
    } else {
      console.log('登录失败', res.errMsg);
    }
  }
});
```

## 后端 code2Session 接口

```
GET https://api.weixin.qq.com/sns/jscode2session?appid=APPID&secret=SECRET&js_code=CODE&grant_type=authorization_code
```

返回:
```json
{
  "openid": "oXXXX-xxxxxxxxxxxx",
  "session_key": "xxxxxxxxxxxxxx",
  "unionid": "oYYYY-yyyyyyyyyyyy",   // 满足条件时返回
  "errcode": 0,
  "errmsg": "ok"
}
```

## wx.checkSession

检查 session_key 是否过期:

```javascript
wx.checkSession({
  success() {
    // session_key 未过期，登录态有效
  },
  fail() {
    // session_key 已过期，需要重新 wx.login
    wx.login({ /* ... */ });
  }
});
```

## 关键约束

1. **code 只能用一次**: 每次 `wx.login` 返回的 code 只能使用一次，且有效期 5 分钟
2. **session_key 有效期**: 非固定时长，用户长期未使用小程序可能失效
3. **不要在客户端使用 session_key**: `session_key` 应在后端保存，不能传给前端
4. **openid 是用户唯一标识**: 同一用户在不同小程序中的 openid 不同
5. **unionid**: 同一微信开放平台账号下的应用，用户 unionid 唯一。需满足条件才可获取

## 云开发登录

使用云开发时，登录更简单:

```javascript
// 云函数中
exports.main = async (event, context) => {
  const { OPENID, APPID, UNIONID } = cloud.getWXContext();
  
  // OPENID 即为当前用户的 openid
  // 无需手动换取
  return { openid: OPENID };
};
```

客户端调用:
```javascript
wx.cloud.callFunction({
  name: 'login',
  success(res) {
    console.log('openid:', res.result.openid);
  }
});
```

## UnionID vs OpenID

| | OpenID | UnionID |
|---|--------|---------|
| 范围 | 单个小程序 | 同一开放平台下的所有应用 |
| 唯一性 | 每个小程序不同 | 跨应用唯一 |
| 获取条件 | wx.login 即可 | 需将小程序绑定到开放平台 |

## 常见错误

| 错误码 | 说明 |
|--------|------|
| 40029 | code 无效（已使用或过期） |
| 45011 | 频率限制，每个用户每分钟 100 次 |
| -1 | 系统繁忙，稍后重试 |
