能力
描述
WeUI组件库
基于样式库 weui-wxss 开发的小程序扩展组件库

kbone 是一个致力于微信小程序和 Web 端同构的解决方案
框架扩展
小程序提供框架扩展能力
工具类库
微信小程序 API 的 TypeScript 类型定义文件

微信同声传译插件是微信自研的语音输入，文本翻译等功能的插件封装，用于提供给第三方小程序调用。
自然语言理解
自然语言理解小程序插件集成微信AI自研的自然语言理解能力
