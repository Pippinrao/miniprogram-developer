小程序无障碍设计指南
#
一、为什么要做无障碍
为了帮助残障人士更加平等便捷地获取、使用互联网应用信息，我们建议开发者结合自身用户群体与使用场景，对小程序进行无障碍适配。
#
二、如何开发无障碍功能
对于界面由微信绘制的接口组件（如
wx.showModal
、
navigation-bar
，微信已经内置了一些无障碍的特性。
对于非原生界面内容，开发者可以添加无障碍标签进行开发，详细信息可参考
aria-component
。
