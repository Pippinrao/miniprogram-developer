# 接口文档

来源: https://developers.weixin.qq.com/miniprogram/dev/framework/device/voip-plugin/api/initByCaller.html

#
initByCaller(Object req)
本接口为异步接口，返回
`Promise`
对象。
发起通话并获取通话房间号。调用此接口后，会创建 VoIP 房间，并且向接听方推送接听提醒。
建议先阅读
接口介绍
。

#
返回值
由于历史原因，本接口调用失败可能会抛出异常（一般是参数错误），也可能会返回
`isSuccess: false`
（一般是后台错误）。
接口成功返回后，仍需要使用
`isSuccess`
字段判断调用是否最终成功。

#
示例代码

```
const wmpfVoip = requirePlugin('wmpf-voip').default

try {
  const { isSuccess } = await wmpfVoip.initByCaller({
    caller: {
      id: '拨打方 Id',
      name: '拨打方名字',
    },
    listener: {
      id: '接听方 Id',
      name: '接听方名字',
    },
    roomType: 'video',
    businessType: 1,
    voipToken: 'xxxx*****xxxx',
    miniprogramState: 'developer', // 开发阶段建议使用开发版
  })

  if (isSuccess /* && 当前不在插件页面 */) {
    wx.redirectTo({
      url: wmpfVoip.CALL_PAGE_PATH,
    })
  } else {
    wx.showToast({
      title: '呼叫失败',
      icon: 'error',
    })
  }
} catch (e) {
  wx.showToast({
    title: '呼叫失败',
    icon: 'error',
  })
}

```
